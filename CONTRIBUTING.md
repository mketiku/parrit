# Contributing to Parrit 🦜

First off, thank you for considering contributing to Parrit! It's people like you that make Parrit such a great tool for the engineering community.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct. We expect all contributors to be respectful and collaborative.

## How Can I Help?

- **Reporting Bugs**: If you find a bug, please [open a GitHub Issue](https://github.com/mketiku/parrit/issues).
- **Suggesting Features**: Have an idea for a new feature? We'd love to hear it! Open an issue to discuss it.
- **Submitting Pull Requests**:
  1. Fork the repository.
  2. Create a new branch for your feature or bugfix.
  3. Ensure your code follows the project's style (TypeScript, Prettier/ESLint).
  4. Write tests for your changes where applicable.
  5. Submit a PR and describe your changes in detail.

## Development Setup

1. Follow the local setup instructions in the [README.md](README.md).
2. Run `npm install` to get dependencies.
3. Run `npm run dev` to start the development server.
4. Run `npm run test` to ensure all tests pass before submitting.
